# E-Learning Management System

## Tech Stack
- Spring Boot
- MySQL
- Spring Security
- JPA
- Maven

## Features
- User Registration & Login
- Role Based Access (Admin / Instructor / Student)
- Course Creation
- Enrollment System

## How to Run
1. Clone the project
2. Configure MySQL in application.properties
3. Run ElearningApplication.java
4. Test APIs using Postman
